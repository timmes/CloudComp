Getting Started

System Requirements

  • Modern web browser (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)
  • JavaScript enabled
  • Local storage enabled (for data persistence)
  • No server or installation required

First Time Setup

  1. Open index.html in your browser
  2. The application will automatically initialize with empty data
  3. Start by importing existing data or creating teams manually
  4. Data automatically saves to your browser's local storage

Tip: Bookmark the application for quick access. All your data persists between sessions.

Importing Data

Course Activity Files

Upload Excel (.xlsx) or CSV files with course completion data. The importer auto-detects the format.

ColumnRequiredDescription
EmailYesUser's email address
Course IDYesUnique course identifier
Course TitleYesName of the course
StatusYesCOMPLETED or IN PROGRESS
LevelNofundamental / intermediate / advanced / specialty
CourseTypeNoAWS Cloud Quest, Digital Course, etc.
Completed onYes*Completion date (for completed courses)
ScoreNoQuiz score (0–100)

Classroom Training (ILT) Files

ILT files are auto-detected by their format: Name, Email, Status columns without a Course ID.

ColumnDescription
NameAttendee's full name
EmailAttendee's email address
StatusCompleted / Not Completed

Classroom training awards 100 points per session.

Teams Meeting Files

CSV exports from Microsoft Teams. The parser extracts meeting titles and attendee emails. Each attendee receives 25 points per meeting.

Import Process

  1. Go to the Import tab
  2. Select course activity files or Teams meeting files
  3. Click Process All Files
  4. Review the import log for any warnings
  5. Check the dashboard for updated statistics

Note: The system automatically skips duplicate entries based on user email + course/meeting ID.

Managing Users

Users are automatically created when importing activity data. Each user is identified by their email address (lowercased).

User Properties

PropertyDescription
EmailPrimary identifier (lowercase)
NameFrom import data or derived from email
Total PointsDerived from all activities
Month PointsDerived from current month's activities
TeamOptional team assignment

Bulk Operations

  1. Select users via checkboxes
  2. Choose an action: Award Points, Assign to Team, or Export Selected
  3. Fill in the details and confirm

Manual Points

Click Add Points to award custom points for special achievements. Enter the user's email, activity title, and point value.

Managing Teams

Creating Teams

  1. Navigate to the Teams tab
  2. Click Create Team
  3. Enter name, description, and choose a color

Managing Members

Click Manage on any team to add or remove members. Team points are automatically derived from member activity totals.

Tip: A user can belong to one team at a time. Assigning to a new team automatically removes them from the previous one.

Bulk Operations

  • Adjust Points — Add, subtract, multiply, or set values across selected activities
  • Delete Selected — Remove activities (points are automatically recalculated)
  • Export Selected — Download selected activities as JSON

Point Configuration

Go to the Configuration tab to adjust point values for each activity type. Changes apply to future imports only — existing activities keep their original points.

Configuration Management

  • Save Changes — Persist current values
  • Reset Defaults — Restore original point values
  • Export Config — Download as JSON

Campaigns

Campaigns let you track learning progress within a specific time window for a defined set of participants.

Creating a Campaign

  1. Go to the Campaigns tab
  2. Click Create Campaign
  3. Set name, dates, and link teams or individual participants
  4. Save as Draft, then Activate when ready

Participant Resolution

Linking a team automatically includes all its members as campaign participants. Individual users can also be added directly.

Leaderboards

Each campaign has its own individual and team leaderboards, scoped to activities within the campaign's date window from eligible participants only.

Status Lifecycle

StatusDescription
draftBeing configured, not yet tracking
activeTracking activities within the date window
completedEnd date passed, results finalized
archivedHidden from active views

Reports & Exports

Available Reports

  • Leaderboard Report — Current month user rankings with points
  • Activity Report — Complete list of all activities
  • Summary Report — Overall statistics, top users, and top teams

Data Export

Click Export in the header to create a complete JSON backup of all users, teams, activities, campaigns, and configuration.

Monthly Reset

  1. Go to the Reports tab
  2. Click Reset Monthly Leaderboard
  3. The system archives current standings as a JSON download

Tip: Export data regularly as a backup. The file is timestamped for easy organization.

Tips & Best Practices

Daily

  • Check the dashboard for a quick performance overview
  • Award manual points for special achievements promptly

Weekly

  • Import new activity data
  • Keep team memberships current

Monthly

  • Generate and share leaderboard reports
  • Reset monthly points after announcing winners
  • Export a full data backup

Troubleshooting

IssueSolution
Data not savingCheck browser settings → Privacy → ensure localStorage is enabled
Import not workingVerify file has required columns (Email, Course ID, Title, Status)
Points incorrectCheck Configuration tab settings and save changes
Teams not showing pointsEnsure members are assigned via the Manage option

Important: Never clear browser data without exporting a backup first. Clearing site data will permanently delete all stored information.
